Einfügen in einen Splay-Baum

Das Einfügen eines Knotens in einen Splay-Baum verläuft analog zum Einfügen eines Knotens in einen binären Suchbaum. Sobald der neue Knoten eingefügt wurde wird er mittels der Splay-Operation zur neuen Wurzel des Baumes.

Suchen in einem Splay-Baum

Die Suche in einem Splay-Baum verläuft analog zur Suche in einem binären Suchbaum. Sobald der gesuchte Knoten gefunden ist wird er mittels der Splay-Operation zur neuen Wurzel des Baumes.

Löschen in einem Splay-Baum

Das Löschen eines Knotens in einen Splay-Baum verläuft analog zum Löschen eines Knotens in einen binären Suchbaum. Nachdem der Knoten aus dem Baum entfernt wurde wird der Vater des Knotens mittels Splay-Operation zur neuen Wurzel des Baumes.

Splay-Operation

Splay(T,x)
if x == T.root
	return;
elseif x.p == T.root
	if x == x.p.left
		rotateRight(root)
	else
		rotateLeft(root)
else
	p = x.p
	g = p.p
	if x == p.left AND p == g.left
		rotateRight(g)
		rotateRight(p)
	elseif x == p.right AND p == g.right
		rotateLeft(g)
		rotateLeft(p)
	elseif x == p.right AND p == g.left
		rotateLeft(p)
		rotateRight(g)
	elseif x == p.left AND p == g.right
		rotateRight(p)
		rotateLeft(g)
splay(T, x)
